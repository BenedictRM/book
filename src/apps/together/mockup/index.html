App Name

Ways and Means

Collaborative Activity Type

The goal of this app is to simulate an election--originally conceived for another game I was developing on my own (though never implemented). The basic functionality of the app would be to elect players/users into various roles for a political simulator (e.g. President, Senator etc.) each of which will have varying powers. For this app however powers will just be listed as strings rather than implemented.

Another idea is to allow players regions be visible on a map and simulating something similar to a typical electorate. If time permits, players will be able to launch campaigns before voting occurs

Ideal Number of Users

10-100-- A cap of 100 players should limit game runtimes to something reasonable

Necessary User Skills

Users wouldn't need too many skills beyond a basic understanding of U.S. elections processes and access to (and ability to) use computers. The game will likely be timed as well so users would need to be responsible to time limits in the voting

Activity Type (e.g. fun)

The primary activity type for this app is for fun/playing a game

How Can a Person View Current Activity?

The app will likely use firebase to store users/election results. Due to thiss data storage method, it will be easy to pull results live into an app and display live election results. This could initially be done by just displaying results with text and as the app progresses it might be possible to display results using a map that changed colors depending on how results are occuring. An early idea would be to limit the game to two parties and two candidates that everyone votes on to limit the scope. This would make it easier to display the map.

How Can Users View Results

When the time limit has expired the user simply has to view the same page as the live results--though updates would cease at this pointe, and perhaps a card or popup would inform users of the winner.

What Can Admin See

Admin can see everything users can see, as well as who is currently online. Perhaps admin could also grant time extensions as needed for games.
